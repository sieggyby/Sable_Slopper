# Thumbnail Headline Prompt Guide

Guide for Claude when writing thumbnail headlines for the Sable clip pipeline.

---

## Format Rules

- **4–6 words** — the sweet spot for readability at thumbnail scale
- **ALL CAPS renders best** — the CSS does `text-transform:uppercase`, but write the headline in ALL CAPS anyway so the preview matches the output
- **No punctuation** — no commas, periods, question marks, colons, or dashes
- **No hashtags** — they clutter the frame and test poorly
- **5th-grade vocabulary** — short Anglo-Saxon words hit harder than jargon

---

## Structures That Work

| Structure | Example |
|-----------|---------|
| Bold claim | `ALGORITHMS BEAT HARDWARE EVERY TIME` |
| Curiosity gap | `THE TRICK BIG TECH HIDES` |
| Stakes framing | `ONE MONOPOLY TO RULE ALL` |
| Contrarian hook | `WHY DATA DOESN'T MATTER` |
| Number hook | `THREE WAYS CRYPTO WINS` |
| "They" framing | `WHAT THEY DON'T TELL YOU` |

---

## Per-Palette Color Guidance

Match the emotional register of the headline to the palette:

| Palette | Emotion | Best for |
|---------|---------|----------|
| `red` | Danger, controversy, urgency | Regulatory risk, market crashes, conflict |
| `blue` | Authority, trust, depth | Macro analysis, institutional moves, policy |
| `purple` | Power, money, prestige | VCs, whales, protocol governance, power dynamics |
| `orange` | Energy, disruption, speed | New launches, builder culture, momentum |
| `green` | Growth, opportunity, upside | Bull signals, adoption curves, winning bets |
| `teal` | Tech, futurism, precision | Infrastructure, ZK proofs, protocol design |

---

## Crypto / Web3 Signal Words

These test well in crypto-native audiences — use when relevant:

- **Power/conflict**: `monopoly`, `cartel`, `insider`, `rigged`, `captured`
- **Systems**: `incentive`, `algorithm`, `governance`, `mechanism`, `protocol`
- **Risk**: `collapse`, `exploit`, `contagion`, `rug`, `cascade`
- **Opportunity**: `asymmetric`, `underpriced`, `unlock`, `catalyst`, `rotation`
- **Identity**: `decentralized`, `permissionless`, `sovereign`, `on-chain`

---

## What to Avoid

- **Questions** — `IS CRYPTO DEAD?` tests 15–20% lower CTR than equivalent statements
- **Vague superlatives** — `AMAZING`, `GREAT`, `INCREDIBLE`, `HUGE` — say what's amazing
- **Unexplained acronyms** — `ZK SNARK EVM` means nothing at a glance; `ZERO KNOWLEDGE WINS` lands
- **Passive voice** — `MARKETS ARE BEING MOVED` → `WHALES MOVE MARKETS`
- **More than 6 words** — font drops to 72px and lines wrap; punchy is better

---

## Quick Reference

```
Good:  ALGORITHMS BEAT HARDWARE EVERY TIME   (5 words, bold claim, orange)
Good:  ONE MONOPOLY TO RULE ALL              (5 words, stakes, purple)
Good:  THE TRICK BIG TECH HIDES             (6 words, curiosity gap, red)
Good:  WHY INCENTIVES ALWAYS WIN            (4 words, contrarian, blue)

Bad:   IS THIS THE END OF CRYPTO?           (too long, question, vague)
Bad:   AMAZING NEW DEVELOPMENTS IN WEB3     (vague superlatives, jargon)
Bad:   MARKETS ARE EXPERIENCING VOLATILITY  (passive, 4+ syllable words)
```
